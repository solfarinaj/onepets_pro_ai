# Modelo de Negocio - OnePets

## Problema
Las veterinarias pequeñas suelen gestionar sus pacientes de forma manual o con scripts básicos, lo que dificulta el seguimiento histórico y la búsqueda rápida de datos.

## Solución (MVP)
Un sistema web CRUD desarrollado en Django que permita:
1.  Registrar pacientes con datos del dueño y la mascota.
2.  Controlar el peso y estado de esterilización.
3.  Búsquedas instantáneas por nombre de mascota o dueño.

## Usuarios Objetivos
- **Veterinarios:** Personal que registra y consulta los datos técnicos.
- **Dueños de Mascotas:** Los beneficiarios finales del seguimiento correcto.

## Diferenciador Pro
Este proyecto no es solo un CRUD; está diseñado con **Ingeniería de Contexto** y **Testing Automatizado** para asegurar la calidad del software.
