# OnePets Pro - Sistema de Gestión Veterinaria

## Punto de Entrada
Este documento sirve como índice maestro para la documentación de ingeniería de contexto del proyecto OnePets.

## Contexto del Proyecto
OnePets es un sistema CRUD migrado de un script lineal a Django, diseñado para gestionar pacientes en clínicas veterinarias pequeñas.

- **Base de Datos:** SQLite
- **Frontend:** HTML + Bootstrap 5
